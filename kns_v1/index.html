Product Spotlight

Power Monitor

An ideal solution for protecting your precision equipment, medicine, and perishable food items in any air-conditioned environment. The Power Monitor helps you ease the burden of monitoring by providing real-time data on environmental conditions, keeping you informed of undesirable temperature changes or power failures.

Key Features

  • ✓ SMS alerts for AC power failure/recovery
  • ✓ Temperature & Humidity monitoring
  • ✓ Easy configuration via SMS
  • ✓ IoT backend communication ready
  • ✓ Built-in 3.7v backup battery
  • ✓ Relay outputs for visual/audible alarms

Applications

  • • Cool Rooms
  • • Chillers & Freezers
  • • Pharmaceutical Storage Facilities
  • • Any temperature-regulated facility
